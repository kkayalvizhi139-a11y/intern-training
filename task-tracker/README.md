# Task Tracker

## Overview

Task Tracker is a browser-based web application built using HTML, CSS, JavaScript, and TypeScript.

It allows users to:

- Add tasks
- Set priority
- Set due date
- Mark tasks as Done
- Filter tasks
- Sort tasks
- Store tasks using Local Storage
- View task summary
- View priority summary

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TypeScript

---

## Features

- Add Task
- Mark Task as Done
- Undo Completed Task
- Clear All Tasks
- Filter Tasks
- Sort by Priority
- Sort by Due Date
- Live Task Counter
- Local Storage Persistence
- Task Summary Table
- Priority Summary Table
